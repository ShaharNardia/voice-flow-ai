# PWA icons

Required files for the PWA to install cleanly:

- `icon-192.png` (192×192)
- `icon-512.png` (512×512)
- `icon-maskable-512.png` (512×512, safe zone inside an 80% centered circle)

Suggested content: red-filled rounded square background (`#F22F46`) with the
white `PhoneOff` icon used elsewhere in the app.

Until these are added, browsers will install the PWA with a generic default
icon — the app still works, just looks slightly off on the home screen.
